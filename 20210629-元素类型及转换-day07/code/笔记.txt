2.元素分类
    常规分类：
        块级元素：可以设置宽高，独占一整行
        行级元素：不可以设置宽高，有多宽占据多宽.
        行内块元素：可以设置宽高，有多宽占据多宽。

        元素嵌套的规则：
            行级元素包行级元素，块级元素可以包裹行级和块级元素。
            a可以包裹块级元素。
            p标签里面不能有块级元素。

    非主流元素分类：
        置换元素：浏览器根据元素的标签和属性，决定显示的内容。  img input button
        非置换元素：除去置换之外的元素，就是非置换。 
        
3.元素类型的转换
    display:block | inline-block | none | inline;
        block：显示为块
        inline-block：显示为行内块
        none：隐藏
        inline：显示为行级元素

    鼠标悬停在e身上就是   e:hover{   }
