0.昨日回顾
    1.自适应
        概念：不同的设备、分辨率，有不一样的排版布局。
        宽度自适应
            1.不设置宽度
            2.设置百分比
            3.通过设置最小最大宽度
                min-width
                max-width
            版心

        高度自适应
            1.不设置高度
            2.设置百分比
            3.通过设置最小最大高度
                min-height
                max-height
        
        假定盒子里面的子元素有浮动
            浮动会让元素脱离文档流
            所以外层盒子里面相当于没有元素，所以不设置高度的情况下，他就高度为0
            所以需要清除浮动所带来的影响，简称清除浮动
                方式1： 
                        给浮动元素的父亲设置 overflow:hidden;
                方式2；
                    给浮动元素最后添加一个空的兄弟div,添加一个类名clear,设置样式  .clear{ clear:both; }
                方式3：
                    给浮动元素的父亲添加类名clearfix，设置样式  .clearfix:after{ content:''; display:block; clear:both;height:0;  }

        注意：如果使用百分比设置高度，html和body记得要设置100%；
    伪元素
        e:after{ content:"" }  在e元素的内容后面添加东西

1.昨日补充的重点
    1.上边固定，下边自适应
        上边和下边都设置大小和绝对定位。
        下边不设置高度，且下边多设置一个 bottom:0; top:上边盒子的高度;
    
    2.上下固定，中间自适应
        上边和下边中间都设置大小和绝对定位。
        中间不设置高度，且中间多设置一个 bottom:下边的高度; top:上边盒子的高度;
    
    3.左边固定，右边自适应
        左边设置宽，设置左浮动
        右边设置overflow:hidden;
    
    4.左边固定，右边固定，中间自适应
        左边中间设置固定宽度，左边左浮动，中间右浮动
        中间设置overflow:hidden;
